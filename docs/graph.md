# Graph

Document Graph API integrated with `var`:

- `graph(n)` creation and `graph()` constructor
- `add_node`, `add_edge`, `set_node_data`, `get_node_data`
- Traversals: `dfs`, `bfs`
- Algorithms: `get_shortest_path`, `bellman_ford`, `floyd_warshall`, `prim_mst`
- Serialization: `to_dot`, `save_graph`
- How Graphs are represented in `var` (`GraphPtr`) and `isGraph()` checks

Examples: building and querying a small graph.
